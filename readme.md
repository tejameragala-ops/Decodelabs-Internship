# 🤖 Rule-Based AI Chatbot

A beginner-friendly Artificial Intelligence project built during my DecodeLabs Internship.  
This project demonstrates how a chatbot can simulate human conversation using **rule-based logic**, **conditional statements**, and **continuous interaction loops** in Python.

---

## 📌 Project Overview

This chatbot responds to predefined user inputs such as greetings, questions, and exit commands.

It is designed to strengthen core programming concepts including:

- Control Flow
- Decision Making
- Conditional Logic
- User Input Handling
- Loop Execution

---

## 🚀 Features

✅ Responds to greetings (`hi`, `hello`, `hey`)  
✅ Answers basic conversational questions  
✅ Handles AI-related queries  
✅ Runs continuously until the user exits  
✅ Case-insensitive input handling
